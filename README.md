# Hospital booking appointment and consultation system
 -This applications enables users to book appointments and cosultations sessions and get feedback on there bookings

 ## Features
 - Book apppointments
 - View available slots
 - Cancel appointments 
 - User sign-up system
 - Book consultation sessions

 ## Technologies used 
 - React 
 - Javascsript
 - HTML
 - External CSS API

## Installation
1. Clone the repository
2. Open the project folder
3. Run: 

npm install

4. Start the project

npm run dev
